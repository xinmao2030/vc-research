# VC Research — 创投企业投资分析系统 · 安装指南

> 输入企业名，自动输出 7 层结构化投研报告（企业画像 → 融资轨迹 → 投资依据 → 产业趋势 → 估值分析 → 风险矩阵 → 投资建议）

---

## 系统要求

| 项目 | 要求 |
|------|------|
| 系统 | macOS 12 (Monterey) 或更高版本 |
| 芯片 | Apple Silicon (M1/M2/M3/M4) 或 Intel |
| 磁盘 | 约 8GB 可用空间（含 AI 模型） |
| 网络 | 安装过程需联网（安装后可离线使用标杆案例） |

## 安装步骤（约 5-15 分钟）

### 第 1 步：打开 DMG

双击收到的 `VC-Research-x.x.x-Installer.dmg` 文件，会看到：

```
📄 README.txt           ← 简要说明
📂 vc-research-src      ← 程序源码
🖥️ 安装 VC Research.command  ← 双击这个！
```

### 第 2 步：运行安装器

**双击「安装 VC Research.command」**

> ⚠️ 如果 macOS 提示"无法打开，因为它来自身份不明的开发者"：
> 右键点击 → 选择「打开」→ 弹窗中点击「打开」

安装器会自动完成以下 7 个步骤：

| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | Xcode Command Line Tools | 编译工具（可能弹出安装窗口，点「安装」） |
| 2 | Homebrew | macOS 包管理器 |
| 3 | Python 3.12 | 程序运行环境 |
| 4 | pango / cairo | PDF 渲染依赖 |
| 5 | VC Research 源码 | 安装到 ~/vc-research |
| 6 | Python 虚拟环境 | 隔离的依赖环境 |
| 7 | Ollama + Qwen3:8b | 本地 AI 模型（约 5GB） |

**安装过程中可能需要：**
- 输入 Mac 密码（安装 Homebrew 时）
- 等待 Xcode CLT 安装窗口（点击「安装」后等待几分钟）
- 等待 AI 模型下载（约 5GB，取决于网速）

### 第 3 步：安装完成

安装完成后会自动：
- ✅ 在桌面创建「VC Research」应用图标
- ✅ 在浏览器中打开 Dashboard

## 日常使用

### 方式 1：双击桌面图标（推荐）

双击桌面上的 **「VC Research」** 图标：
- 自动启动 AI 引擎 (Ollama)
- 自动打开浏览器 Dashboard
- 在 Dashboard 中输入企业名即可生成研报

### 方式 2：终端命令

打开「终端」App，输入：

```bash
# 分析标杆企业（秒出结果，不需要联网）
vcr analyze "影石创新"

# 分析任意企业（需要 Ollama 运行，首次约 2 分钟）
vcr analyze "字节跳动" --live

# 查看 6 家内置标杆案例
vcr list-examples

# 生成 PDF 研报
vcr analyze "银诺医药" --pdf

# 打开 Web Dashboard
vcr-dashboard
```

## 常见问题

### Q: 安装器双击没反应？
右键点击「安装 VC Research.command」→ 选择「打开」→ 弹窗中点击「打开」

### Q: 提示 "Permission denied"？
打开终端，运行：
```bash
chmod +x ~/Downloads/安装\ VC\ Research.command
```
然后重新双击。

### Q: Xcode CLT 安装弹窗关不掉？
在弹窗中点击「安装」，等待 2-5 分钟完成。如果已经关闭，直接按回车继续。

### Q: Homebrew 安装需要密码？
输入你的 Mac 登录密码（输入时不会显示字符，这是正常的），然后按回车。

### Q: AI 模型下载太慢？
Qwen3:8b 约 5GB，首次下载取决于网速。如果中断，重新运行安装器会继续下载。
也可以跳过模型下载，先使用 6 家标杆案例（无需 AI 模型）。

### Q: 桌面图标双击没反应？
1. 确认 Ollama 已安装：打开终端输入 `ollama --version`
2. 手动启动：打开终端输入 `vcr-dashboard`

### Q: 想更新到新版本？
打开终端，运行：
```bash
cd ~/vc-research && git pull
source activate.sh
pip install -e .
```

### Q: 想卸载？
```bash
rm -rf ~/vc-research
rm -rf ~/.vc-research
rm -f ~/Desktop/VC\ Research.app
rm -rf ~/Applications/VC\ Research.app
brew uninstall ollama  # 可选：卸载 AI 引擎
```

## 研报内容说明

每份研报包含 7 个模块，采用闯关式解锁：

```
🏢 模块 1 · 企业画像    → 创始人背景、核心产品、标志性客户
💰 模块 2 · 融资轨迹    → 每轮融资金额/估值/投资方详情
🎯 模块 3 · 投资依据    → 团队评分、市场规模、护城河分析
🌊 模块 4 · 产业趋势    → 赛道周期、竞争格局、政策环境
💎 模块 5 · 估值分析    → 多方法交叉验证公允估值区间
⚠️ 模块 6 · 风险矩阵    → 现金跑道、监管、竞争、技术风险
🎯 模块 7 · 投资建议    → 参投/观望/回避 + 条款 + 退出路径
```

投资裁决含义：
- **强烈参投** — 综合评分 ≥ 70，高度看好
- **参投** — 综合评分 ≥ 55，值得投入
- **观望** — 综合评分 ≥ 40，需进一步验证
- **回避** — 综合评分 < 40，风险过高

> ⚠️ 本工具仅供学习研究使用，不构成投资建议。
