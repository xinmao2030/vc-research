# 创投术语表 · Glossary

给零基础投资者的一本"行话词典"。每个术语附**类比**,降低记忆成本。

---

## 💰 融资与股权

| 术语 | 英文 | 一句话解释 | 类比 |
|------|------|------------|------|
| 轮次 | Funding Round | 天使→A→B→C→Pre-IPO 的进阶路径 | 游戏通关 |
| 估值 | Valuation | 投资人和创始人谈妥的"公司定价" | 房屋挂牌价 |
| 投前估值 | Pre-money | 融资**前**公司值多少 | 空房价 |
| 投后估值 | Post-money | Pre-money + 新融资额 | 空房价 + 装修款 |
| 稀释 | Dilution | 新钱进来,老股东持股比例变小 | 蛋糕切分 |
| ESOP | Employee Stock Option Pool | 留给员工的期权池(通常 10-15%) | 公司内部奖池 |
| 领投 | Lead Investor | 本轮出资最多 / 定价的主要投资方 | 餐桌买单者 |
| 跟投 | Follow-on | 跟着领投一起进的小股东 | 拼团成员 |
| 优先清算权 | Liquidation Preference | 破产/被收购时,谁先拿回本金 | 救生艇优先级 |
| 反稀释 | Anti-dilution | 后续融资估值下降时,早期投资人的保护条款 | 保价险 |
| 对赌 | VAM | 业绩不达标要赔钱/回购的条款 | 军令状 |

## 📊 市场与增长

| 术语 | 一句话解释 | 类比 |
|------|------------|------|
| TAM | Total Addressable Market - 理论最大市场 | 整个海洋 |
| SAM | Serviceable Addressable - 你能触达的细分 | 能游到的海域 |
| SOM | Serviceable Obtainable - 3-5 年现实份额 | 能抓到的鱼 |
| CAGR | 年复合增长率 | 利滚利增速 |
| MAU / DAU | 月/日活跃用户 | 健身房月卡/日签到 |
| ARR | Annual Recurring Revenue - 年化订阅收入 | 每年雷打不动的租金 |
| GMV | 交易总流水 | 收银台滚动总额 |

## 💎 单位经济学

| 术语 | 一句话解释 | 健康线 |
|------|------------|--------|
| LTV | 单个客户生命周期收入 | 越高越好 |
| CAC | 获客成本 | 越低越好 |
| LTV/CAC | 客户终身价值 ÷ 获客成本 | **≥ 3 健康** |
| 回本周期 | Payback Months - 几个月赚回 CAC | < 12 月优秀 |
| 毛利率 | 收入 - 直接成本 的占比 | SaaS 70%+ |
| 烧钱速度 | Burn Rate - 每月净亏损 | — |
| 跑道 | Runway - 现金够烧几个月 | < 6 月濒死 / 18 月+ 安全 |

## 🏰 商业属性

| 术语 | 解释 | 例子 |
|------|------|------|
| 护城河 | Moat - 让对手难以复制的竞争优势 | 微信网络效应 / 台积电制程 |
| 网络效应 | 用户越多,产品越值钱 | 微信 / 滴滴 |
| 规模效应 | 产能越大,单位成本越低 | 京东物流 |
| 切换成本 | 用户换掉你的代价 | SAP / Salesforce |
| 品牌心智 | "想起赛道=想起你" | 可口可乐 / 特斯拉 |

## 📈 估值方法

| 方法 | 什么时候用 | 类比 |
|------|------------|------|
| 可比公司法 | 赛道有上市对标 | 隔壁小区挂牌价 |
| 可比交易法 | 赛道最近有并购 | 最近成交价 |
| DCF | 现金流可预测 | 未来租金折现 |
| VC 逆推法 | 早期公司没利润 | 退出价倒推入场价 |
| 最近一轮锚点 | 刚融完资不到 6 个月 | 最新挂牌价 |

## ⚠️ 风险与退出

| 术语 | 解释 |
|------|------|
| J 曲线 | VC 基金前期亏损后期盈利的回报形态 |
| DPI | 已分配给 LP 的资金 ÷ LP 出资额 |
| TVPI | 账面价值 + 已分配 ÷ LP 出资额 |
| 基石投资人 | IPO 前就锁定仓位的大型机构 |
| 锁定期 | Lock-up - IPO 后一段时间不能卖 |
| 道琼斯指数 / 恒生 / 纳斯达克 | 主要退出市场 |
